# Quick Start Guide

Get up and running with Khandoba Secure Docs in 5 minutes.

---

## What You'll Learn

- Create your first vault
- Upload a document
- Generate an AI voice report
- Share with a nominee
- Understand basic security features

---

## Step 1: Create Your First Vault (2 minutes)

### Choose Vault Type

**Single-Key Vault:**
- Password protected
- Quick access
- Good for: Personal documents, photos

**Dual-Key Vault:**
- Requires ML approval
- Enhanced security
- Good for: Sensitive documents, business files

### Create Vault

1. Tap **"+"** button (top right)
2. Enter vault name (e.g., "Personal Documents")
3. Choose vault type
4. Set password (if single-key)
5. Tap **"Create"**

**Result**: Your first vault is created and ready for documents!

---

## Step 2: Upload Your First Document (1 minute)

### Upload Options

**From Photo Library:**
1. Open your vault
2. Tap **"+"** button
3. Select **"Photo Library"**
4. Choose a photo or document
5. Tap **"Upload"**

**Take Photo:**
1. Tap **"+"** button
2. Select **"Camera"**
3. Take photo
4. Tap **"Use Photo"**

**Record Voice Memo:**
1. Tap **"+"** button
2. Select **"Voice Memo"**
3. Tap record button
4. Tap stop when done

**Result**: Document is encrypted and stored securely!

---

## Step 3: Generate AI Voice Report (1 minute)

### What is a Voice Report?

A narrated security briefing that explains:
- What's in your vault
- Security status
- Threat analysis
- Recommended actions

### Generate Report

1. Open your vault
2. Tap **"Intelligence"** tab
3. Tap **"Generate Voice Report"**
4. Wait 10-30 seconds
5. Tap **"Play"** to listen

**Result**: You'll hear a voice memo explaining your vault's security status!

---

## Step 4: Share with Nominee (1 minute)

### What is a Nominee?

A trusted person who can access your vault (with your permission).

### Invite Nominee

1. Open your vault
2. Tap **"Sharing"** tab
3. Tap **"Add Nominee"**
4. Select contact from your contacts
5. Choose access level:
   - **Full Access**: All documents
   - **Subset Access**: Selected documents only
6. Tap **"Send Invitation"**

**Result**: Nominee receives invitation and can accept to access your vault!

---

## Step 5: Explore Security Features

- Identify unusual access patterns

**How to View:**
1. Open your vault
2. Tap **"Security"** tab
3. Tap **"Access Map"**

### Threat Monitoring
- Real-time threat detection
- ML-based anomaly detection
- Automatic alerts

**How to View:**
1. Open your vault
2. Tap **"Security"** tab
3. View **"Threat Dashboard"**

## Key Concepts

### Vaults
- Encrypted containers for documents
- Can be single-key or dual-key
- Each vault has its own password/approval

### Documents
- Files stored in vaults
- Automatically encrypted
- AI-tagged and organized

### Nominees
- Trusted users with vault access
- Can be full or subset access
- Transfer ownership available

### AI Features
- Voice reports
- Auto-tagging
- Threat detection
- Source/sink classification

## Common Tasks

### Upload Multiple Documents
1. Open vault
2. Tap **"+"** button
3. Select multiple files
4. Tap **"Upload"**

### Search Documents
1. Open vault
2. Tap search bar
3. Type keywords
4. Results appear instantly

### Delete Document
1. Swipe left on document
2. Tap **"Delete"**
3. Confirm deletion

### Lock Vault
1. Tap vault settings
2. Tap **"Lock Vault"**
3. Vault locks immediately

## Tips for Success

### Organize with Multiple Vaults
- Create separate vaults for different purposes
- Example: "Personal", "Work", "Medical"

### Use Dual-Key for Sensitive Files
- Enhanced security for important documents
- ML automatically approves safe access

### Generate Regular Voice Reports
- Stay informed about security status
- Listen while multitasking

### Share Selectively
- Only nominate trusted users
- Use subset access for limited sharing
