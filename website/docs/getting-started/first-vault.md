# Creating Your First Vault

Learn how to create and configure your first vault in Khandoba Secure Docs.

---

## What is a Vault?

A **vault** is an encrypted container for your documents. Think of it like a digital safe deposit box:
- **Secure**: All documents are encrypted with AES-256
- **Organized**: Keep related documents together
- **Controlled**: You decide who has access

---

## Vault Types

### Single-Key Vault

**Best for:**
- Personal documents
- Photos and memories
- Quick access needed

**Features:**
- Password protected
- Instant access after authentication
- Simple and fast

**Security:**
- Password required
- Biometric authentication (Face ID/Touch ID)
- Session timeout after inactivity

### Dual-Key Vault

**Best for:**
- Sensitive business documents
- Legal files
- Medical records
- Financial documents

**Features:**
- ML-based auto-approval
- Enhanced security
- Access logging
- Geographic monitoring

**Security:**
- ML analyzes each access request
- Automatic approval for safe access
- Manual review for suspicious activity
- Complete audit trail

---

## Creating a Vault

### Step-by-Step

1. **Open App**
   - Launch Khandoba Secure Docs
   - You'll see the vault list (empty if first time)

2. **Tap Create Button**
   - Look for **"+"** button (top right)
   - Or tap **"Create Your First Vault"** if empty

3. **Enter Vault Name**
   - Choose a descriptive name
   - Examples: "Personal Documents", "Work Files", "Medical Records"
   - You can change this later

4. **Add Description (Optional)**
   - Helpful for organization
   - Example: "Important personal documents and photos"

5. **Choose Vault Type**
   - **Single-Key**: Password protected
   - **Dual-Key**: ML approval required

6. **Set Security**
   - **Single-Key**: Create password
   - **Dual-Key**: Configure ML settings

7. **Create Vault**
   - Tap **"Create"**
   - Vault appears in your list

## Vault Configuration

### Basic Settings

**Name**
- Change anytime
- Make it descriptive
- Helps with organization

**Description**
- Optional details
- Useful for multiple vaults
- Helps identify purpose

**Type**
- Cannot change after creation
- Choose carefully
- Create new vault if needed

### Security Settings

**Session Timeout**
- Default: 30 minutes
- Auto-locks after inactivity
- Adjustable in settings

**Biometric Authentication**
- Face ID or Touch ID
- Required for vault access
- Can be disabled (not recommended)

**Access Logging**
- Automatic for all vaults
- Tracks all access events
- Viewable in Security tab

## Organizing Multiple Vaults

### Best Practices

**By Purpose**
- Personal Documents
- Work Files
- Medical Records
- Financial Documents

**By Sensitivity**
- Public (low security)
- Private (standard security)
- Confidential (dual-key)

**By Person**
- My Documents
- Shared with Spouse
- Family Archive

### Tips

- **Limit Vaults**: Too many can be overwhelming (5-10 recommended)
- **Clear Names**: Use descriptive names
- **Regular Review**: Archive or delete unused vaults

## Vault Management

### Opening a Vault

1. Tap vault in list
2. Authenticate (Face ID/Touch ID or password)
3. Vault opens
4. Session starts (30 minutes)

### Locking a Vault

**Automatic:**
- After 30 minutes of inactivity
- When app is backgrounded
- When device locks

**Manual:**
1. Tap vault settings
2. Tap **"Lock Vault"**
3. Vault locks immediately

### Deleting a Vault

⚠️ **Warning**: This permanently deletes all documents in the vault!

1. Tap vault settings
2. Scroll to bottom
3. Tap **"Delete Vault"**
4. Confirm deletion
5. Enter password to confirm

## Vault Sharing

### Adding Nominees

1. Open vault
2. Tap **"Sharing"** tab
3. Tap **"Add Nominee"**
4. Select contact
5. Choose access level
6. Send invitation

### Transferring Ownership

1. Open vault
2. Tap **"Sharing"** tab
3. Tap **"Transfer Ownership"**
4. Select nominee (must be accepted nominee)
5. Confirm transfer

## Troubleshooting

### Can't Create Vault
- **Check Subscription**: Ensure premium subscription is active
- **Check Storage**: Ensure device has free space
- **Restart App**: Close and reopen

### Vault Won't Open
- **Check Authentication**: Ensure Face ID/Touch ID is enabled
- **Check Password**: Verify password is correct
- **Check Session**: Vault may have timed out

### Vault Missing
- **Check Sync**: Ensure iCloud sync is enabled
- **Check Account**: Ensure you're signed in
- **Check Filters**: Vault may be filtered out
