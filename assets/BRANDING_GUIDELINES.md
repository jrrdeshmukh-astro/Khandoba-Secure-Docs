## üé≠ Logo Usage

### Logo Variations

#### Primary Logo
- **Format:** SVG + PNG
- **Colors:** Full color (primary + secondary)
- **Usage:** Main branding, marketing materials

#### White Logo
- **Format:** SVG + PNG
- **Colors:** White
- **Usage:** Dark backgrounds, dark mode

#### Dark Logo
- **Format:** SVG + PNG
- **Colors:** Dark gray/black
- **Usage:** Light backgrounds, light mode

#### Icon Only
- **Format:** SVG + PNG
- **Size:** Square (1:1 aspect ratio)
- **Usage:** App icons, favicons, small spaces

### Logo Clear Space
- **Minimum clear space:** 1x the height of the logo
- **Never:** Crop, rotate, or distort the logo
- **Never:** Place on busy backgrounds
- **Never:** Use colors outside brand palette

## üñºÔ∏è Icon Design

### App Icon Design Principles

1. **Simplicity:** Recognizable at 16x16 pixels
2. **Clarity:** No fine details that disappear at small sizes
3. **Contrast:** High contrast for visibility
4. **Branding:** Uses brand colors
5. **Uniqueness:** Distinctive from competitors

### Icon Elements
- **Primary:** Shield or lock (security)
- **Secondary:** Document or vault door
- **Colors:** Primary coral red (#E74A48) with cyan accent (#11A7C7)
- **Style:** Modern, clean, professional

### Icon Don'ts
- ‚ùå No transparency
- ‚ùå No text (except in specific contexts)
- ‚ùå No photos or complex images
- ‚ùå No gradients that don't work at small sizes
- ‚ùå No copyrighted material

## üé¨ Marketing Materials

### Screenshot Guidelines

#### Content
- Show key features clearly
- Use real app screens (not mockups)
- Highlight unique features (Intel Reports, Dual-Key)
- Include premium subscription screen

#### Style
- Clean, professional appearance
- Consistent with app design
- No overlays or annotations (unless required by store)
- High quality (no pixelation)

### App Preview Video

#### Structure
1. **Hook (0-3s):** "Military-Grade Security"
2. **Features (4-15s):** Show key features
3. **Value (16-27s):** "Unlimited for $5.99/month"
4. **CTA (28-30s):** App name and download

#### Text Overlays
- Font: San Francisco Bold
- Color: White (#FFFFFF)
- Shadow: Black, 20% opacity
- Position: Centered
- Animation: Fade in/out (0.2s)

## üö´ Brand Don'ts

### Colors
- ‚ùå Don't use colors outside the brand palette
- ‚ùå Don't use pure black (#000000) - use #141414
- ‚ùå Don't use pure white (#FFFFFF) on pure white backgrounds

### Typography
- ‚ùå Don't use decorative fonts
- ‚ùå Don't use more than 3 font sizes in one design
- ‚ùå Don't use all caps for body text

### Logo
- ‚ùå Don't modify the logo
- ‚ùå Don't use logo on busy backgrounds
- ‚ùå Don't stretch or distort the logo
- ‚ùå Don't use logo smaller than minimum size

### General
- ‚ùå Don't use stock photos that don't match brand
- ‚ùå Don't use clip art or generic icons
- ‚ùå Don't mix design styles
- ‚ùå Don't use low-quality images
