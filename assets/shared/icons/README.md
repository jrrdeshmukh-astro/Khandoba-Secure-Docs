## ğŸ¨ Design Guidelines

### Icon Design Principles
1. **Simplicity:** Recognizable at small sizes
2. **Consistency:** Same style across all icons
3. **Clarity:** High contrast, clear shapes
4. **Branding:** Uses brand colors
5. **Scalability:** Works from 16px to 1024px

### Color Usage
- **Primary:** #E74A48 (Coral red)
- **Secondary:** #11A7C7 (Cyan)
- **Tertiary:** #E7A63A (Amber)
- **Neutral:** #8E8E93 (Gray)

### Style Guidelines
- **Stroke Width:** 2-3px for outlines
- **Corner Radius:** 4-8px for rounded corners
- **Padding:** 10-20% clear space around icon
- **Grid:** Align to 8px or 16px grid
