## ğŸ¯ Logo Variations

### Primary Logo
- **Format:** SVG (vector) + PNG (raster)
- **Size:** 1024x1024 (PNG)
- **Colors:** Full color (primary + secondary)
- **Usage:** Main branding, marketing materials
- **Location:** `logos/logo-primary.svg` and `logo-primary.png`

### White Logo
- **Format:** SVG (vector)
- **Colors:** White (#FFFFFF)
- **Usage:** Dark backgrounds, dark mode
- **Location:** `logos/logo-white.svg`

### Dark Logo
- **Format:** SVG (vector)
- **Colors:** Dark gray/black (#141414)
- **Usage:** Light backgrounds, light mode
- **Location:** `logos/logo-dark.svg`

### Icon Only
- **Format:** SVG (vector) + PNG (raster)
- **Size:** 512x512 (PNG)
- **Aspect Ratio:** 1:1 (square)
- **Usage:** App icons, favicons, small spaces
- **Location:** `logos/logo-icon.svg` and `logo-icon.png`

## ğŸ“ Wordmarks

### Horizontal Wordmark
- **Format:** SVG (vector)
- **Layout:** Icon + text (horizontal)
- **Usage:** Headers, navigation, wide spaces
- **Location:** `wordmarks/wordmark-horizontal.svg`

### Vertical Wordmark
- **Format:** SVG (vector)
- **Layout:** Icon + text (vertical)
- **Usage:** Sidebars, narrow spaces
- **Location:** `wordmarks/wordmark-vertical.svg`

### Icon Wordmark
- **Format:** SVG (vector)
- **Layout:** Icon with text below
- **Usage:** Marketing materials, presentations
- **Location:** `wordmarks/wordmark-icon.svg`

## ğŸ¨ Design Guidelines

### Logo Usage
- **Clear Space:** Minimum 1x logo height on all sides
- **Minimum Size:** 24px height (for digital use)
- **Never:** Crop, rotate, or distort
- **Never:** Use on busy backgrounds
- **Never:** Use colors outside brand palette

### Color Usage
- **Primary Logo:** Use on white/light backgrounds
- **White Logo:** Use on dark backgrounds (#1F2430, #252C39)
- **Dark Logo:** Use on light backgrounds (#F5F2ED, #FFFFFF)
