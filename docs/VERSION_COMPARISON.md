## Version Numbering

### Format

`MAJOR.MINOR.PATCH`

- **MAJOR** - Breaking changes
- **MINOR** - New features, backward compatible
- **PATCH** - Bug fixes

### Build Number

- Incremental build number
- Used for internal tracking
