# Website Documentation Files

This directory contains the HTML files and documentation needed to upload Terms of Service and Privacy Policy to khandoba.org.

## Files

1. **terms-of-service.html** - Complete Terms of Service page for website
2. **privacy-policy.html** - Complete Privacy Policy page for website
3. **WEBSITE_UPLOAD_GUIDE.md** - Step-by-step guide for uploading files to your website

## Quick Start

1. Read `WEBSITE_UPLOAD_GUIDE.md` for detailed instructions
2. Upload `terms-of-service.html` to `https://khandoba.org/terms`
3. Upload `privacy-policy.html` to `https://khandoba.org/privacy`
4. Verify both URLs are accessible
5. Update App Store Connect with the new URLs

## URLs

- **Terms of Service:** `https://khandoba.org/terms`
- **Privacy Policy:** `https://khandoba.org/privacy`

## Email Addresses

Ensure these email addresses are configured:
- **Legal:** `legal@khandoba.org`
- **Privacy:** `privacy@khandoba.org`

- Both HTML files are self-contained and ready to upload
- Files are mobile-responsive
